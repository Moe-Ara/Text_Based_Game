# How to run the game

1- Clone the repo:
```
git clone https://github.com/Moe-Ara/Text_Based_Game.git
```

2- Go to a folder with the name "PrimaryServices":

```
cd repo_path_on_your_local_machine/PrimaryServices
```

3- build the project and then run it:

```
dotnet build
dotnet run
```
